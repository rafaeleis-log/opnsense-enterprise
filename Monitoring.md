# 📊 Monitoramento

## Objetivo

Garantir alta disponibilidade da infraestrutura por meio de monitoramento contínuo.

---

## Ferramentas

- Zabbix
- Syslog
- SNMP

---

## Itens monitorados

### Firewall

- CPU
- Memória
- Disco
- Interfaces
- Temperatura

### VPN

- Status
- Latência
- Disponibilidade

### Rede

- Gateways
- Links
- Switches
- Access Points

### Servidores

- Windows Server
- Active Directory
- DNS
- DHCP

---

## Alertas

- Link indisponível
- Gateway Offline
- VPN desconectada
- Uso elevado de CPU
- Uso elevado de memória

---

## Benefícios

- Resposta rápida
- Prevenção de falhas
- Histórico de desempenho
- Planejamento de capacidade
